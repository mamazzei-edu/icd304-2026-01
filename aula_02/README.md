**Exercício 1**


Calcule $\frac{\partial f}{\partial x}$ e $\frac{\partial f}{\partial y}$ para a função:$$f(x, y) = 5x^7 + 4y^4 - 15$$

Gabarito Passo a Passo:

Em relação a $x$: O termo $4y^4$ e o número $-15$ são constantes. A derivada de uma constante sozinha é zero.

$$\frac{\partial f}{\partial x} = 35x^6 + 0 - 0 = \mathbf{35x^6}$$

Em relação a $y$: Agora, o termo $4x^4$ e o $-15$ são as constantes e desaparecem. O $3$ do expoente desce multiplicando o $4$.

$$\frac{\partial f}{\partial y} = 0 + 12y - 0 = \mathbf{12y}$$
